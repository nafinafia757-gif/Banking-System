# Banking-System

 Banking System (Python Project):
 
 Project Overview:
 
This project is about creating a simple Banking System using Python.
The program allows users to login and perform basic banking operations.
It includes features like deposit, withdrawal, and balance checking.

Login :

The user enters a account number and password.
The program checks if the details are correct.
If correct, login is successful.
If wrong, the program will stop.

Program Structure:

The program is created using classes and functions.
Two classes are used:

Deposit

Withdraw

Both classes have the same method called action, but they perform different tasks.

Banking Operations:

Deposit:
The user can add money to the account.
The balance will increase.
Withdraw:
The user can withdraw money from the account.
If balance is not enough, an error message is shown.

 Balance Check:
 
The user can view the current balance at any time.

Exit:

The user can exit the program.

 Concepts Used:
 
Classes and Objects
Functions
Loops
Conditions
Polymorphism

Polymorphism:

Both Deposit and Withdraw classes use the same method name action.
But they perform different operations.
A common function is used to call them.
This is called polymorphism.

 Program Flow:
 
User logs in
Menu is displayed
User selects an option
Operation is performed
Menu repeats until exit

Key Points:

Simple and easy to use
Menu-driven program
Uses basic and advanced Python concepts
Demonstrates polymorphism clearly

Conclusion:
This project helped in understanding Python basics and object-oriented concepts.
It is a simple application for learning banking operations.
